## Solr入门 
<div class="content-intro view-box ">Solr 使程序员能够轻松地开发具有高级功能的复杂、高性能的搜索应用程序。它是一个独立的企业级搜索应用服务器，它对外提供 API 接口。用户可以通过 HTTP 请求或者使用 Http Get 提出查找请求。
